\#Mystic Notes



Un site simplu pentru iubitorii de parfumuri autentice. Explorează parfumuri după tip,vezi colecții și creează propria ta aromă.



\## Cum se deschide



Deschide `index.html` în browser. Nu e nevoie de instalare.
